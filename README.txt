# Fast New Quota  
This mod speeds up the quota animation in Lethal Company.  
Now you don't have to wait for slow animations!  

## Installation  
1. Install [BepInExPack](https://thunderstore.io/package/BepInEx/BepInExPack_LethalCompany/).  
2. Download the mod and extract it into your `BepInEx/plugins` folder.  
3. Enjoy faster quota animations!  

## Configuration  
After first launch, a config file will be created in `BepInEx/config/com.pxntxrez.FastNewQuota.cfg`.  
You can change `QuotaSpeedMultiplier` to adjust the speed.
